# 📊 NexSent – AI Sentiment Insight Analyzer

NexSent is an AI-powered sentiment analysis tool that allows users to upload ANY CSV file and automatically analyze text sentiment using Natural Language Processing.

---

## 🚀 Features

- Upload any CSV file
- Auto-detect text columns
- AI sentiment classification (Positive, Neutral, Negative)
- Polarity scoring
- Interactive dashboard with charts
- Download analyzed dataset
- AI-generated insights report

---

## 🧠 Technologies Used

- Python
- Streamlit
- Pandas
- TextBlob (NLP)
- Matplotlib & Seaborn
