### BizIntel AI 📊

AI-Powered Sentiment Analysis & Business Insights Platform

## 📌 Overview

BizIntel AI is a full-stack AI-powered business analytics platform that enables users to upload structured datasets and instantly generate:

Sentiment analysis
Business insights
Statistical summaries
Interactive visualisations
Downloadable reports
AI-powered Q&A responses

The platform is designed for business analysts, researchers, students, and decision-makers who want to extract insights from business data without coding.

🚀 Features

✅ User Authentication System
✅ Forgot Password Recovery
✅ OTP Verification
✅ CSV / Excel / JSON Upload Support
✅ AI Sentiment Analysis
✅ Interactive Data Visualisations
✅ Dynamic Dashboard
✅ Business Insight Reports
✅ Executive Summaries

✅ PDF & DOCX Report Export

⚙️ Installation
1. Clone Repository
git clone https://github.com/sellwane7/bizintel_ai.git
cd bizintel_ai
2. Create Virtual Environment
Windows
python -m venv venv
venv\Scripts\activate
macOS / Linux
python3 -m venv venv
source venv/bin/activate
3. Install Dependencies
pip install -r requirements.txt
4. Run Application
streamlit run app.py
